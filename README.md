# Particles Generator

A generator of particles, useless but fun..

[particles.pruvostbastien.fr](https://particles.pruvostbastien.fr)


## Features

- 4 particles layers
- Change quantity
- Change opacity
- Change size
- Change speed
- Change color


## Run locally

1. Clone the project:

```bash
  git clone https://github.com/bastien-pruvost/particles-generator.git
```

2. Go to the project directory:

```bash
  cd particles-generator
```

3. Run `index.html` file with your browser
